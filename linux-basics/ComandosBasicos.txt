Navegacion
pwd -> ruta actual
ls  -> ver contenido del directorio actual
cd  -> movers

Creaacion
mkdir -> crear carpeta
mkdir -p home/main/proyecto -> crea toda la ruta si no existe
touch archivo.txt -> solo crea o actualiza fecha
cat archivo.txt -> ve todo el contenido
nano archivo.txt -> crea el archivo y permite modificarlo

Copiar, mover y borrar
cp origen destino -> copia 
mv archivo.txt carpeta/ -> mueve el archivo a la carpeta
rm archiv.txt -> borra el archivo
rm -r carpeta/ -> borra carpeta y contenido de la misma

Busqueda
find /ruta -name archivo.txt -> ejemplo: find /var/log -name "*.log"

Permisos
3 entidades: usuario(owner), grupo, otros

permisos: leer r 4, escribir w 2, ejecutar x 1

ejemplo -> chmod 755 script.sh

Ejecutar archivos
./script.sh -> en caso de ver permission denied falta permiso x o ejecutar

Cambiar dueño
chown usuario archivo
chown usuario:grupo archivo

ejemplo ->  chown andre:devs app.log

Usuarios y Grupos

id usuario -> para saber toda la informacion del usuario


ver usuario actual -> whoami id
crear usuarios -> 
sudo useradd andre
sudo passwd andre

userdel juan
userdel -r juan   # borra home

cat /etc/passwd -> ver usuarios existentes 
sudo chsh -s /bin/bash juan -> cambia el shell de manera segura

Crear grupo -> sudo grupoadd devs
borrar grupo -> groupdel developers
agregar usuario a grupo -> sudo usermod -aG devs andre
sin -a sacarías al usuario de otros grupos
cambiar grupo primario -> usermod -g developers juan
cat /etc/group -> ver grupos existentes

whoami       # usuario actual
who          # usuarios conectados
w            # sesiones activas
last         # historial de logins
su - usuario # cambiar usuario
sudo comando # ejecutar como root

cat /etc/shells => ver shells permitidos
